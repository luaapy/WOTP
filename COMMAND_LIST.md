## üéØ **QUICK TIPS**

1. **All commands are case-insensitive**
   - `|MENU`, `|menu`, `|MeNu` all work the same

2. **Use |search to find commands**
   - `|search otp` - Find OTP-related commands
   - `|search game` - Find game commands

3. **Commands without prefix**
   - `halo`, `hi`, `hello`, `test` work without `|`

4. **Get category help**
   - Type `|category_name` to see all commands in that category

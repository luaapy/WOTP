Documentation

Learn how to integrate the WOTP API into your applications. All requests must be authenticated using a Bearer Token.

Authentication

Acquire your API key from the Dashboard settings. Header format:

Authorization: Bearer YOUR_API_KEY

Endpoints

POST /api/v1/otp/send

Sends a one-time password to a WhatsApp number.

{ "phone": "6283834946034", "message": "Kode OTP Anda adalah {{code}}", "code": "123456", "expiryMinutes": 5 }

POST /api/v1/otp/verify

Verifies the code sent to a specific number.

{ "phone": "6283834946034", "code": "123456" }

GET /api/v1/otp/status

Returns the platform's current connection status.

Rate Limiting

Standard accounts are limited to 100 requests per hour. If you require higher limits, please contact administration.
